# Project Skills

Claude Code skills for building mobile-optimized Three.js games with
Entity Component Systems (ECS), React Three Fiber, and TypeScript.

Each subdirectory is a skill (`<name>/SKILL.md`) discovered automatically by
Claude Code in web/CLI/IDE sessions. Invoke one manually with `/<skill-name>`.

## Categories
- **threejs** — scene setup, geometry, materials, textures, lighting, shadows,
  post-processing, particles, instancing, animation, model loading, more
- **ecs** — architecture, components, systems, queries, events, serialization
- **react** — React Three Fiber setup, patterns, performance, state, ECS integration
- **mobile** — performance, touch input, battery, memory
- **typescript** — game types, ECS types, performance
- **game-systems** — input, collision, physics, AI, audio, UI, camera, spawn,
  health/combat, inventory, level/progression
